ğŸ—ï¸ Technology Stack
This project is architected using the MERN stack, leveraging modern libraries for performance and scalability.

Frontend
React.js (Vite): For building a fast, interactive user interface.

Tailwind CSS: For responsive, utility-first styling.

Context API: For state management across the application.

React Router DOM: For seamless client-side routing.

Backend
Node.js & Express.js: Robust server-side framework for handling API requests.

MongoDB (Mongoose): NoSQL database for efficient data storage (Users, Products, Orders).

Tools & Services

Firebase: For Google Sign-In authentication.

Multer: Middleware for handling file uploads.

Render: Cloud platform used for deployment.
